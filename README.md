# ✈️ PHAK Reader - Aviation Study App

A modern web application for studying the FAA Pilot's Handbook of Aeronautical Knowledge (PHAK).

## 🚀 Features

- **Complete PHAK Handbook** - All 17 chapters
- **Study Tools** - Highlighting, notes, bookmarks
- **496 Glossary Terms** - Searchable aviation vocabulary
- **Search Everything** - Full-text search across content, glossary, and notes
- **Night Mode** - Dark theme for late-night studying
- **Practice Quizzes** - Test your knowledge
- **Interactive Trainers** - Simulations and tools
- **ACS Integration** - Airman Certification Standards references

## 🛠️ Tech Stack

- **React 18** - Modern UI framework
- **Vite** - Fast build tool
- **Zustand** - State management with persistence
- **React Router** - Client-side routing

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌐 Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically

### Manual Build

```bash
npm run build
# Output in /dist folder
```

## 📚 Data Sources

- **PHAK Content** - FAA-H-8083-25C
- **Glossary** - 496 aviation terms extracted from FAA glossary
- **ACS Codes** - FAA-S-ACS-6C Private Pilot standards

## 🎯 Usage

1. **Read** - Navigate through chapters with sticky sidebars
2. **Highlight** - Select text and choose from 5 colors
3. **Take Notes** - Add notes with custom tags
4. **Bookmark** - Save important pages
5. **Search** - Find anything across all content
6. **Study** - Use quizzes and interactive trainers
